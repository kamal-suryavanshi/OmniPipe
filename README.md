# OmniPipe Core

A generic, production-ready CG Studio Pipeline core designed for distributed teams.

## Features
- Fully Decoupled Architecture
- Context-Driven API
- Typer-based CLI interface
- Agnostic Asset Management (Kitsu enabled by default)
- **Zero-Install (Vendorized Dependencies)**

## Getting Started

Because all dependencies are tracked internally, you do not need PIP or Poetry!

```bash
# Clone the repository
git clone https://github.com/kamal-suryavanshi/OmniPipe.git
cd OmniPipe

# Run immediately using standard Python!
python -m omnipipe --help
```
